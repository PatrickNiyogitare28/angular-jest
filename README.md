# Angular Jest
E2E and Unite tests with Jest in angular

## Environment
- Node
- Angular CLI grobally installed on dev environment


## Getting started
- [X] clone the [Repo](https://github.com/PatrickNiyogitare28/angular-jest)
- [X] run `yarn install` or `npm install`

## Running tests
- `npx jest` or `ng test`
